# RECUPERA NOTAS
Projeto em [Ruby](https://www.ruby-lang.org/pt/) para recuperar notas via site.

### PRE-REQUISITOS
- Instalar o ruby

### DEPENDENCIAS
- Executar os comandos

    ```yml
    bundle init
    bundle install
    ```

### EXECUCAO
- Executar os comandos
    ```yml
    rake 'read_notes'
    ```
